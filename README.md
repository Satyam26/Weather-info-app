# Weather-info-app

### Prerequisites
 ---
Install `NodeJs` and `npm`

### Setup  and Deployment
---

- copy or clone repo, and do `npm install` or `yarn install`. 


-  Navigate to `http://localhost:3000/`. The app will automatically reload if you change any of the source files.
